### Android Studio를 통해서 앱을 만들어보았습니다.


<img width="507" height="216" alt="image" src="https://github.com/user-attachments/assets/75b037d7-be72-47bb-b91b-a4a8e10034a8" />



#### 목적 

다양한 레시피를 간편하게 제공하여 요리고민 해결


재료를 기반으로 가능한 요리를 추천


레시피를 저장하고 관리할 수 있는 기능 제공


#### 기능 

홈 화면 : 추천 레시피와 검색 기능 제공

레시피 검색 : 재료별/카테고리별 검색

레시피 상세보기 : 요리 과정, 재료, 사진 표시

즐겨찾기 : 관심 레시피 저장 및 관리

내 레시피 : 사용자가 직접 레시피 추가

#### 주요 기능 소개 및 사용 가이드 


<img width="584" height="594" alt="image" src="https://github.com/user-attachments/assets/e405304f-342c-4a15-8a77-1c90aec5736a" />


홈화면에서 레시피 보기를 누르면 오른쪽과 같이 이전에 저장했던 레시피들이 나온다.

<img width="528" height="602" alt="image" src="https://github.com/user-attachments/assets/6d8e9800-a5d9-40cc-a70c-d30dc52d9fe7" />

홈화면에서 레시피 추가 버튼을 누르거나

홈화면에서 레시피 확인 버튼을 누르고 레시피 추가 버튼을 누르면


레시피를 추가할 수 있는 액티비티가 나오게 했다.


여기서 요리 이름, kcal, 재료, 레시피를 입력받게 했다.


갤러리에서 추가하기 버튼을 누르면 갤러리에서 사진을 가져올 수 있게했다



<img width="559" height="604" alt="image" src="https://github.com/user-attachments/assets/b998c885-f535-4545-8334-f1319b51385c" />


왼쪽은 갤러리에서 사진을 가져온 모습이다.


추가가 다 끝나고 뒤로가기를 누르면 앞서 저장했던 내용이 나오도록 했다


<img width="587" height="599" alt="image" src="https://github.com/user-attachments/assets/d5466649-ea7b-4e24-bc6e-12ef9451c987" />


레시피를 꾹 누르면 레시피를 삭제할 수 있다.


또 레시피를 클릭하면 레시피 정보들을 자세히 볼 수 있다.
